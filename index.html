Introduction

Traffic congestion is a growing problem in urban areas worldwide, resulting in significant economic losses, environmental pollution, and reduced quality of life. Traditional traffic signal systems operate on fixed timing cycles that do not adapt to changing traffic conditions, leading to inefficient traffic flow and unnecessary delays.

Our Smart Traffic Signal Control System leverages cutting-edge artificial intelligence to optimize traffic flow at intersections. Using YOLOv8, a state-of-the-art object detection model, we analyze real-time traffic conditions from multiple directions and dynamically adjust signal timings to minimize congestion and improve overall traffic efficiency.

The system processes video feeds or images from four directions (North, South, East, West), detects vehicles using advanced computer vision techniques, and calculates optimal green light durations based on vehicle density. This adaptive approach ensures that traffic signals respond intelligently to changing traffic patterns, reducing wait times and improving road safety.

Objectives

The primary objectives of this research are:

  • To develop a real-time vehicle detection system using computer vision
  • To implement an adaptive traffic signal control mechanism that dynamically optimizes signal timings
  • To reduce congestion by minimizing vehicle waiting times
  • To improve fuel efficiency and reduce emissions through optimized traffic flow
  • To create a user-friendly interface for traffic analysis and signal timing recommendations

Methodology

System Architecture

The system consists of four main components:

  1. Data Collection Module: Gathers traffic videos/images from four directions
  2. Vehicle Detection Module: Uses YOLOv8 to detect and count vehicles
  3. Signal Control Module: Implements adaptive timing algorithm
  4. User Interface Module: Provides web-based interaction and visualization

Vehicle Detection

The YOLOv8 model was selected after comparative analysis with other object detection models. YOLOv8 provides optimal balance of speed and accuracy for real-time applications:

  • Model: YOLOv8n (nano variant for speed)
  • Vehicle classes: car, bus, truck, motorcycle, bicycle
  • Confidence threshold: 0.25 (optimized for traffic scenarios)
  • Processing: Apple MPS acceleration for real-time performance
  • Inference time: 20-40ms per frame on Apple M1 hardware

System Architecture Diagram

The above diagram illustrates the complete system architecture, showing the flow from data collection through vehicle detection to adaptive signal control and user interface.

Our methodology follows a structured approach where traffic data from four directions (North, South, East, West) is collected and processed through the YOLOv8 detection module. The detected vehicle counts are then fed into the adaptive signal control algorithm, which calculates optimal green times. The entire process is visualized through a user-friendly web interface, enabling real-time monitoring and analysis.

Adaptive Signal Control Algorithm

The adaptive control algorithm calculates green time based on vehicle density:

Green_Time = max(10, min(60, Base_Time + (k √ó Vehicle_Count)))
  • Base_Time = 15 seconds (minimum viable green time)
  • k = 2 seconds per vehicle (scaling factor)
  • Minimum limit = 10 seconds (safety constraint)
  • Maximum limit = 60 seconds (fairness and efficiency constraint)

The direction with the highest vehicle count receives priority, ensuring that the busiest approach gets the longest green time allocation.

Conclusion

Key achievements include:

  • Successful implementation of real-time vehicle detection with 95%+ accuracy
